# 앱인토스 Metro 설정 - 아주 쉽게 설명

## 먼저: Metro가 뭐예요?

**Metro = 지금 만드는 앱 코드를 보여주는 "개발용 TV 방송국"** 같은 거예요.

- 샌드박스 앱 = TV  
- Metro = 그 TV가 받아오는 방송 (우리가 만드는 앱 코드)

그래서 **Metro를 먼저 켜놓아야** 샌드박스 앱이 우리 앱을 볼 수 있어요.

---

## 한 줄 요약

> **1단계: Metro 켜기** → **2단계: 폰이 Metro에 연결되게 하기** → **3단계: 샌드박스 앱에서 스킴 입력**

---

## 전체 순서 (3단계)

### 1단계: Metro 켜기 (항상 먼저!)

1. 맥에서 **터미널**을 연다.
2. 프로젝트 폴더로 이동:
   ```bash
   cd /Users/eldrac/Desktop/bamboo
   ```
3. 아래 명령어 실행:
   ```bash
   npm run dev
   ```
4. 터미널에 `Metro`, `8081`, `5173` 같은 글씨가 보이면 **켜진 거**예요.
5. **이 터미널 창은 닫지 마세요!** Metro가 켜져 있어야 합니다.

---

### 2단계: 폰/시뮬레이터가 Metro에 접속하게 하기

**어떤 기기를 쓰는지에 따라 다릅니다.**

#### 🍎 iPhone 시뮬레이터 (맥에서만)

- **별도 설정 없음**  
- Metro가 켜져 있으면 시뮬레이터는 자동으로 연결돼요.

---

#### 📱 iPhone 실기기 (진짜 아이폰)

1. **맥과 아이폰이 같은 와이파이**에 연결되어 있어야 해요.
2. 맥에서 터미널을 열고, 내 컴퓨터 IP 주소 확인:
   ```bash
   ipconfig getifaddr en0
   ```
   → 예: `192.168.0.10` 같은 숫자가 나와요.
3. 샌드박스 앱 실행 → 처음에 **"로컬 네트워크"** 허용 메시지가 뜨면 **허용**.
4. 샌드박스 앱에서 **서버 주소를 입력하는 칸**이 보이면, 아까 나온 IP 주소를 입력하고 저장.

---

#### 🤖 Android (휴대폰 또는 에뮬레이터)

1. **USB로 컴퓨터에 연결**해야 해요.
2. 맥 터미널에서 아래 명령어를 실행:

   ```bash
   adb reverse tcp:8081 tcp:8081
   adb reverse tcp:5173 tcp:5173
   ```

3. 정상이면 아무 메시지 없이 끝나요.
4. `adb` 설치가 안 되어 있으면, Android Studio 설치 후 SDK Platform-Tools에 포함된 `adb`를 사용하면 됩니다.

---

### 3단계: 샌드박스 앱에서 앱 실행

1. 앱인토스 샌드박스 앱 실행
2. 로그인 → 앱 선택
3. **스킴 입력 칸**에 아래를 입력:

   ```
   intoss://bamboo-app
   ```

4. **실행** 또는 **열기** 버튼 누르기
5. 화면 상단에 `Bundling ...%` 같은 글이 보이면 **Metro와 연결된 거**예요. 잠시 기다리면 앱이 뜹니다.

---

## 자주 나오는 문제

### "잠시 문제가 생겼어요"

- Metro가 **켜져 있는지** 확인하세요. (`npm run dev` 실행했는지)
- Android라면 `adb reverse` 명령어를 **다시 한 번** 실행해 보세요.

### Metro를 끄고 싶을 때

- Metro가 켜져 있는 터미널에서 **Ctrl + C** 누르면 꺼져요.
