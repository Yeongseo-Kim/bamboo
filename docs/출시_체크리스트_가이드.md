# 앱인토스 비게임 출시 체크리스트 가이드

출시 전 [비게임 출시 가이드](https://developers-apps-in-toss.toss.im/checklist/app-nongame.html)를 검토하기 위한 자동 테스트와 수동 확인 가이드입니다.

## 자동 검증 (테스트 코드)

다음 명령으로 출시 체크리스트 중 **자동 검증 가능한 항목**을 실행할 수 있습니다.

```bash
# 소스 코드만 검증 (번들 없이)
npm run test:checklist

# 번들 빌드 후 전체 검증 (권장: 출시 직전)
npm run checklist:bundle
```

### 검증 항목 요약

| 구분 | 검증 내용 |
|------|----------|
| **접속/스킴** | `intoss://` 스킴 사용, `intoss-private://` 미사용 |
| **TDS** | `@toss/tds-react-native` 사용, PageNavbar 등 내비게이션 컴포넌트 |
| **설정** | granite.config appsInToss, brand.displayName, .granite/app.json |
| **콘텐츠** | 신고/클린봇 등 위법 콘텐츠 필터 관련 코드 |
| **번들** | 번들에 `intoss-private` 미포함, 용량 100MB 이하 |

### 번들 상태 검사 가능 여부

**가능합니다.** `granite build` 후 생성되는 `dist/bundle.android.js`, `dist/bundle.ios.js`를 대상으로 다음을 검사합니다.

- 금지 스킴 `intoss-private://` 포함 여부
- 번들 용량 100MB 이하 (압축 해제 기준)

번들이 없으면 해당 검사는 자동으로 스킵됩니다.

## 수동 확인 필요 항목

아래 항목은 **샌드박스 앱**에서 실제로 실행해 보며 확인해야 합니다.

- [ ] 모든 화면에서 뒤로가기 버튼 정상 동작
- [ ] 최초 화면에서 뒤로가기 시 미니앱 종료
- [ ] 바텀시트가 진입 시 자동으로 열리지 않음
- [ ] 스크롤·터치·화면 전환 등 인터랙션이 2초 이상 지연되지 않음
- [ ] 라이트 모드 테마
- [ ] 사용자 안내/확인 시 TDS 모달 사용
- [ ] 권한 요청 전 사용자 동의 UI
